# Python Loop Examples

This is a collection of simple Python scripts that show how to use loops (`for` and `while`) to count numbers, filter even or odd numbers, and make multiplication tables.

---

## 📄 What Each File Does

### 1. `count.py`

This file shows different ways to count and add up numbers:

* **Counting up:** Prints numbers from 1 to 10.
* **Counting down:** Prints numbers backward from 10 down to 1.
* **Even numbers:** Prints even numbers from 0 to 20.
* **Adding numbers:** You type a number, and it adds up all the numbers from 1 to your number.
* **Big table:** Prints a giant multiplication table from $1 \times 1$ up to $10 \times 10$.

### 2. `even.py`

This file focuses entirely on filtering even and odd numbers:

* Shows two different ways to find even numbers: by skipping numbers (using a step of 2) or by using math to check if a number can be divided evenly by 2.
* Asks you for a number and prints all the odd numbers up to that limit.

### 3. `factorial.py`

This file calculates a **factorial** (which means multiplying a number by every number below it; for example, the factorial of 5 is $5 \times 4 \times 3 \times 2 \times 1 = 120$).

* It does this math problem two ways: once using a `for` loop and once using a `while` loop.

### 4. `multiplication.py`

This file helps you practice times tables:

* **Single table:** Enter a number, and it prints the 1-to-10 times table for that specific number.
* **Custom grid:** Enter how wide and how tall you want a times table grid to be, and it builds it for you.

---

## 🚀 How to Run Them

1. Make sure you have **Python** installed on your computer.
2. Open your computer's terminal or command prompt.
3. Run any file by typing `python` followed by the file name. For example:

```bash
python factorial.py

```
